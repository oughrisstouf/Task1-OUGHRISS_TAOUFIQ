# Task1-OUGHRISS_TAOUFIQ
## OT_FITNESS – Build Your Body · Unlock Strength

**OT_FITNESS** is a modern, responsive fitness landing page designed to promote Coach OT’s holistic training philosophy.  
It combines **mindset coaching**, **functional training**, and **precision nutrition** into a visually engaging web experience.  
The site includes interactive elements such as a bookmarking system, animated progress tracker, newsletter sign‑up, and a transformation roadmap.

## ✨ Features

| Category | Details |
|----------|---------|
| **Hero Section** | Strong call‑to‑action with fitness tracker mockup, dynamic stats, and coach quote. |
| **Fitness Library** | Grid of articles with tags, authors, and a bookmark toggle (local UI state). |
| **Side Panel** | Progress bars (animated on scroll), trending workouts list, newsletter widget, and tag cloud. |
| **Three Pillars** | Mindset, Training, Nutrition – presented with icons, tags, and hover effects. |
| **Transformation Roadmap** | 6‑step interactive timeline with active/hover states and visual badges. |
| **Toolkit (Arsenal)** | Showcases equipment, cardio tools, and nutrition system with gradient cards. |
| **Mobile Menu** | Hamburger navigation with smooth open/close and accessibility support. |
| **Page Loader** | Animated logo and progress bar that disappears after content loads. |
| **Scroll to Top** | Appears after scrolling down 400px – smooth return to top. |
| **Accessibility** | Skip link, ARIA labels, focus outlines, reduced‑motion support, and semantic HTML. |
| **Responsive** | Fully responsive – works on mobile, tablet, and desktop (grid breakpoints at 768px / 1024px). |

## 🛠 Tech Stack

- **HTML5** – Semantic markup, accessibility attributes
- **CSS3** – Custom properties (theming), Flexbox, CSS Grid, animations, media queries
- **JavaScript (Vanilla)** – Intersection Observer, local UI interactions, dynamic class toggles, form handling
- **External Libraries**
  - [Google Fonts](https://fonts.google.com/) – Montserrat & Roboto
  - [Font Awesome 6](https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.5.1/css/all.min.css) – Icons in footer

> No build tools or frameworks – pure static site ready to deploy anywhere.
